# `giternal` Demo Application #
## Getting Started ##
   * install from source (gem is currently broken):
     * `git clone git://github.com/pat-maddox/giternal.git /tmp/giternal`
     * `cd /tmp/giternal`
     * `rake install_gem` or the `sudo` equivalent
     * `cd -`
   * `rake gems:install`
   * update via `giternal update`
